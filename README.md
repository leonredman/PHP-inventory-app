# PHP Inventory App

Application created to assist in the tracking of a Culinary Business Inventory


## Deployment

Application hosted on Heroku with ClearDB MySQL

## Built With

*  - PHP
*  - MySQL
*  - Frontend Framework Bootstrap
*  - Jquery
*  - MAMP
